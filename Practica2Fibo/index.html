Contexto del Problema

Se plantea la necesidad de planificar un ahorro mensual que no sea estático. Utilizando la serie de Fibonacci, el ahorro aumenta progresivamente mes a mes de forma estructurada, permitiendo metas financieras escalables.

Explicación del Algoritmo

El algoritmo calcula la secuencia de Fibonacci iterativamente. Inicializa dos variables, suma sus valores para obtener el siguiente término, y actualiza las variables en cada ciclo, acumulando el monto total sin utilizar vectores.
